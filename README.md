# Battle Blaster (Unreal Engine 5)

A **third-person shooter** built in **Unreal Engine 5** using **C++ and Blueprints**.  
This project demonstrates player combat, shooting mechanics, and health systems.

---

## 🎮 Features
- Player shooting mechanics (ranged combat)  
- Health system with damage and respawn logic  
- Weapon control and ammo management  
- Dynamic HUD updates for next level and restart level 
- Mix of **C++ gameplay code** and **Blueprint scripting**

---

## 🧠 What I Learned
- Implementing **C++ classes** for shooting, health, and weapons  
- Using **Blueprints** for HUD and gameplay flow  
- Handling player input, UI updates, and game mechanics in UE5  
- Debugging and optimizing gameplay logic
